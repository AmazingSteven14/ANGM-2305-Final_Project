## Description
My project is suppose to be a simple arcade like crane game that makes use of the the pygame library to create all of the objects
in my game. The design is pretty simple as I mostly focused on making sure that everything worked correctly over how it looked.
If I had to change anything in the future, I would definitely add some randomness to the movement of my prize since it can only move left and right as of now. I would also try to add more than just one prize to grab since I only have one prize as of right now.
